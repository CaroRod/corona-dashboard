# CORONA DASHBOARD

    This project was build based with node js version 16.9.1

## Deploy Steps

1. Clone repo
   `git clone https://github.com/CaroRod/corona-dashboard.git`
2. Go to project folder
   `cd corona-dashboard`
3. Instal required dependences
   `npm install`
4. Start server
    `npm start`
5. Open browser on `http://localhost:3000`
